Adding some style

All HTML elements have default style. For example the hr tag by default is centered and is 1 pixel thick, and it may be hard to see but it is "inset". By adding our own custom styles we can make things look different than the defaults.

Formally these styles are known as CSS (cascading style sheet), and there are three kinds. Here is a good place to use a list:

  • inline - added as an attribute of a tag
  • embedded - in between <style> tags in the page header
  • linked - a separate file

Note that the closer the instruction is to the HTML tag the higher the instructions priority, while the farther away the more general the instructions become. For website development, linked style is the most powerful since it allows the developer to apply consistent styles to all the pages in a website without repetitious coding.
